# Cannabis Invoicing App

This project is a comprehensive invoicing application designed specifically for the cannabis industry. It allows users to manage invoices, track payments, and support cannabis-related businesses effectively.

## Features
- User authentication
- Invoice management
- Payment tracking
- Reporting tools

## Technologies Used
- Next.js
- TypeScript
- Tailwind CSS

## Getting Started

1. **Clone the repository**
2. **Install dependencies**: `npm install`
3. **Start the development server**: `npm run dev`
